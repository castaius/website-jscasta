# JSCASTA Integrated Starting Plan

This plan connects the JSCASTA business strategy with the systems principles from `Modern Marketing Magic`.

## How The Two Fit Together

The Katrina reference gives the long-term system:

visitor -> inquiry -> CRM -> fast response -> sales call -> proposal -> contract -> payment -> preparation -> performance -> review -> referral/follow-up

The JSCASTA strategy gives the immediate focus:

Sydney corporate roving magic for event organizers.

So the right starting move is not to build everything at once. It is to build the first working version of that journey for one buyer and one offer.

## Current Focus

Primary market:

- Corporate events in Sydney

Primary offer:

- 60-90 minutes of interactive roving magic for corporate receptions, networking events, conferences, awards nights, client functions, and company celebrations.

Primary buyer:

- Event agencies
- Corporate event planners
- Executive assistants
- Marketing managers
- People and Culture managers
- Conference producers
- Hotel and venue event coordinators

Primary CTA:

- Tell me about your event

## What To Build First

### 1. Minimum Viable Corporate Landing Page

The website should answer these questions fast:

- What is this?
- Is it for my type of event?
- What problem does it solve?
- Can I trust him?
- How do I inquire?

Recommended page flow:

1. Hero: corporate roving magic in Sydney, with clear CTA.
2. Outcome: guests connect, conversations start, event energy rises.
3. Video/photo proof.
4. How roving magic works.
5. Suitable event types.
6. Benefits for organizers.
7. Credibility: international experience, selected brands, psychology background.
8. Testimonial or proof.
9. FAQ.
10. Inquiry form.

### 2. Lead Tracker Before Full CRM

Before choosing a full CRM, create a simple tracker with:

- Inquiry date
- Name
- Company
- Role
- Email
- Phone
- Event type
- Event date
- Location
- Guest count
- Lead source
- Quoted amount
- Status
- Next follow-up date
- Notes

This preserves Katrina's most important CRM lesson: data is money.

### 3. Fast Response Workflow

Every inquiry should trigger a fast personal response.

Suggested first workflow:

- Within 1 hour: personal email or call.
- Same day: send a short proposal or next-step email.
- 2 days later: follow up if no response.
- 7 days later: follow up again.
- After event: request testimonial/review.
- 6-12 months later: reactivation follow-up.

This can start manually and later become automated.

### 4. Sales Call Script

The sales call should not sound like selling tricks. It should diagnose the event.

Key questions:

- What kind of event are you planning?
- Who will be in the room?
- What moment in the event needs the most energy?
- Is the goal networking, celebration, client experience, team culture, or something else?
- What would make the entertainment feel successful to you?
- Are there speeches, awards, food service, or other programmed moments I should work around?

Positioning line:

> I use close-up magic as a way to create conversation, energy, and shared moments throughout the room.

### 5. Local Proof Plan

Because local proof is limited, every early Sydney opportunity should create proof.

For each strategic appearance, try to get:

- 5-10 strong photos
- 3-5 short vertical video clips
- 1 written testimonial
- Permission to mention the venue/company if possible
- A referral introduction

Free or discounted appearances should only be used when they produce useful proof, useful relationships, or both.

## What Not To Build Yet

Do not start with:

- A huge blog
- Multiple unrelated offers
- Complex automation
- A full brand overhaul
- A large SEO architecture
- Public pricing
- Too many CTAs

Those come after the first corporate inquiry journey works.

## First 14 Days

Day 1-2:

- Gather best photos, videos, testimonials, logos, and TV/brand proof.
- Sort assets into: strong for Sydney corporate, useful later, not useful now.

Day 3-5:

- Draft the corporate landing page copy.
- Build the inquiry form.
- Create the lead tracker.

Day 6-8:

- Add video/photo proof.
- Add credibility section.
- Add FAQ.
- Make the CTA consistent: "Tell me about your event."

Day 9-10:

- Write response templates.
- Write discovery call questions.
- Write a simple proposal template.

Day 11-14:

- Build a list of 50 Sydney corporate/event targets.
- Start outreach.
- Track replies, calls, and next actions.

## The Practical Starting System

For now, the system is:

1. One focused landing page.
2. One inquiry form.
3. One lead tracker.
4. One response template.
5. One sales call structure.
6. One proposal template.
7. One local proof collection habit.
8. One weekly review of numbers.

That is enough to begin.

## Core Reminder

The old proof is not useless. It just needs to be translated.

Do not ask Sydney buyers to care that JSCASTA was known somewhere else. Show them that he can solve the problem they have right now: making their event feel alive, polished, easy, and memorable.
