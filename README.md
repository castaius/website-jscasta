# Jscasta Website

Static website for Jscasta.

## Open locally

Open `index.html` in a browser.

Spanish version: `index-es.html`.

## Project files

- `index.html` - main English page
- `index-es.html` - Spanish page
- `styles.css` - site styles
- `script.js` - site interactions
- `assets/` - images and static assets
- `jscasta-theme/` - theme files
